# MyLineGroup Corporate Site

One-page B2B website for a company that provides comprehensive office and business supplies.

## Included sections

- Hero with value proposition and CTAs
- About company
- Product categories
- Key advantages
- Work process
- Contacts + request form

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Main files

- `app/page.tsx` - page content and section structure
- `app/globals.css` - global style tokens and base settings
- `app/layout.tsx` - metadata and root layout
